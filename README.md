**EZ Screen Overlay Tool** is a lightweight, always-on-top utility designed for someone that seeks for simple text overlay editing. It allows you to display customizable, transparent text on your screen while recording or presenting—completely eliminating the need to edit text into your videos in post-production.

## ✨ Why Use This Tool?
For simple video that require simple text overlay editing, stop opening Premiere Pro or DaVinci Resolve just to add simple "Step 1, Step 2" text instructions to your tutorial videos. Pre-write your steps, save them as a session, and use global hotkeys to cycle through them live while you record. 

## 🚀 Key Features

* **👻 True Ghost Mode (Click-Through):** The overlay text is completely immune to mouse clicks. You can place text dead-center on your screen and continue clicking and typing in your applications underneath it.
* **🧲 Smart Mini Toolbar:** Launch a detached, floating director's toolbar to control your text without keeping the main app open. Drag it to the top or bottom of your screen to **auto-snap and hide it**, revealing itself only when you hover over the edge.
* **⌨️ Global Hotkeys:** Control the flow of your presentation without losing focus on your primary application.
* **💾 Session Management:** Save different presets for different projects (e.g., "Python Tutorial Session", "Agency Pitch Session") and load them instantly.
* **🎯 Pixel-Perfect Nudging:** Use the built-in D-Pad to fine-tune your text position so it never covers important UI elements in your recordings.
* **🎨 Highly Customizable:** Adjust font size, font colors, background colors, transparency, and auto-hide timers.

## 🎹 Global Hotkeys

You can control the overlay from anywhere on your computer, even when playing a game or using a full-screen application.

| Shortcut | Action |
| :--- | :--- |
| `Ctrl + Shift + S` | **Show** the current text overlay |
| `Ctrl + Shift + H` | **Hide** the current text overlay |
| `Ctrl + Shift + N` | **Cycle** to the next saved preset and display it |
